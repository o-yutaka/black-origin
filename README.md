# BLACK ORIGIN

BLACK ORIGIN is a modular planetary-scale artificial intelligence operating system designed to continuously discover global datasets, build knowledge graphs, run planetary simulations, coordinate agent swarms, and evolve its own architecture.

## Phase-oriented bootstrap

1. Analyze architecture
2. Initialize repository
3. Implement kernel
4. Implement engines
5. Implement simulation
6. Implement UI
7. Implement evolution engines

## Quick start

```bash
make install
make run
```

Then open `http://localhost:8000` for API and `http://localhost:8000/ui` for the control interface.

## Services

- Milvus (vector intelligence)
- Neo4j (knowledge graph)
- Kafka (stream engine)
- Redis (cache/coordination)

Run platform dependencies:

```bash
docker compose up -d
```
